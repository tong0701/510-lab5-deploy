## Lab 5 app (GIX Checkout Dashboard)

Streamlit + Supabase project lives in **`gix-checkout-dashboard/`**. Run locally: `cd gix-checkout-dashboard`, create venv, `pip install -r requirements.txt`, copy `.env.example` to `.env`, then `streamlit run app.py`. Deploy on Streamlit Cloud with main file path **`gix-checkout-dashboard/app.py`** and set secrets `SUPABASE_URL` / `SUPABASE_KEY`. Details in that folder’s `README.md`.
